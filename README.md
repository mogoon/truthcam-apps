# TruthCam Apps

블록체인 기반 사진 진위 검증 서비스 TruthCam의 데스크탑 애플리케이션입니다.

Blockchain-based photo verification desktop applications by TruthCam.

---

## TruthScan

대량의 사진을 한 번에 검증하는 배치 검증 도구입니다.

Batch verification tool for verifying multiple photos at once.

### Features
- 폴더 단위 일괄 검증 / Batch verification by folder
- TruthCam API 연동 / TruthCam API integration
- CSV 내보내기 / CSV export
- 블록체인 검증 결과 확인 / Blockchain verification results
